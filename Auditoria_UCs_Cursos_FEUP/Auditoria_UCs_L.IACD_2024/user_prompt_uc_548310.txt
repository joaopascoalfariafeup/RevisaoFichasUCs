# Ficha a avaliar - versão em português

## Unidade curricular
Tecnologias Web

## Curso responsável
Licenciatura em Ciência de Computadores

## Créditos ECTS
6

## Ano
3

## Semestre
1

## Objetivos
<p>O objetivo desta unidade curricular é familiarizar os alunos com os conceitos e tecnologias utilizados no desenvolvimento de aplicações centradas na web. Isto inclui aprender sobre os diferentes tipos de aplicações web, as diferentes tecnologias que podem ser usadas para construí-las e as melhores práticas para projetar e desenvolver aplicações web.</p>

## Resultados de aprendizagem e competências
<p>Após este curso, os alunos serão capazes de projetar e implementar aplicações centradas na web que sejam escaláveis, seguras e amigáveis ao usuário. Eles poderão fazer isso entendendo os diferentes aspectos do desenvolvimento web, incluindo o back-end do servidor, o front-end do navegador, marcação, apresentação, interação e comunicação assíncrona.</p>

## Pré-requisitos e co-requisitos
Competências razoáveis de programação (pré-requisito)

## Programa
<p>* HTML, XHTML e CSS <br />* JavaScript e manipulação de objetos no browser usando o DOM <br />* Protocolo HTTP <br />* AJAX: comunicação assíncrona entre cliente e servidor <br />* Algumas APIs do HTML5 <br />* Programação de servidores web com node.js<br />* Outros modelos para implementação de servidores web dinâmicos: CGI e alternativas</p>

## Métodos de ensino e atividades de aprendizagem
Nas aulas teóricas é usado o método expositivo, sendo apresentada uma visão organizada dos temas do programa, incluindo exemplos de aplicação das diferentes tecnologias. Nas aulas práticas são resolvidos exercícios e trabalhos práticos de consolidação dos conceitos expostos nas aulas teóricas.

## Tipo de avaliação
Avaliação distribuída sem exame final

## Componentes de Avaliação
- Teste: 60.0 %
- Trabalho prático ou de projeto: 40.0 %

## Componentes de Ocupação
- Estudo autónomo: 54.0 horas
- Frequência das aulas: 48.0 horas
- Trabalho laboratorial: 60.0 horas

## Fórmula de cálculo da classificação final
<p>A avaliação da unidade curricular é distribuída, sem exame final, sendo composta por dois testes teóricos, realizados ao longo do semestre, e um trabalho prático com entrega por etapas a realizar em grupos de até 3 alunos.<br /> <br />A classificação final consiste em 8 valores para trabalho prático e 12 valores para os testes  É necessário obter uma nota mínima de 25% tanto no trabalho como no conjunto dos testes.</p>

## Obtenção de frequência
<p>Serão considerados como tendo frequentado a unidade curricular os estudantes que tenham participado em pelo menos um dos testes teóricos previstos e realizado pela menos uma entrega do trabalho. Além disso, é necessária uma nota mínima de 25% quer no conjunto dos testes, quer no conunto das entregas do trabalho.</p>

## Melhoria de classificação
<p>Dado a avaliação ser distribuída e não ser realizado um exame final, não há lugar a exames de recurso ou de melhoria de nota.</p>



# Ficha a avaliar - versão em inglês

## Unidade curricular
Web Technologies

## Curso responsável
Bachelor in Computer Science

## Créditos ECTS
6

## Ano
3

## Semestre
1

## Objetivos
<p>The goal of this curricular unit is to familiarize students with the concepts and technologies used in the development of web-centered applications. This includes learning about the different types of web applications, the different technologies that can be used to build them, and the best practices for designing and developing web applications.</p>

## Resultados de aprendizagem e competências
<p>After this course, students will be able to design and implement web-centered applications that are scalable, secure, and user-friendly. They will be able to do this by understanding the different aspects of web development, including the server backend, the browser frontend, markup, presentation, interaction, and asynchronous communication.</p>

## Pré-requisitos e co-requisitos
Reasonable programming skills (prerequisite)

## Programa
<p>* HTML, XHTML and CSS<br />* JavaScript and object handling in the browser using DOM<br />* HTTP protocol<br />* AJAX: asynchronous communication between client and server <br />* Some HTML5 APIs<br />* Programming web servers with node.js <br />* Other models for implementing dynamic web servers: CGI and alternatives<br /><br /></p>

## Métodos de ensino e atividades de aprendizagem
<p>Theoretical classes use lectures to present an overview of the syllabus topics and examples of different technologies. Practical classes allow students to work on exercises and assignments that reinforce the concepts learned in theory.</p>

## Tipo de avaliação
Distributed evaluation without final exam

## Componentes de Avaliação
- Teste: 60.0 %
- Trabalho prático ou de projeto: 40.0 %

## Componentes de Ocupação
- Estudo autónomo: 54.0 hours
- Frequência das aulas: 48.0 hours
- Trabalho laboratorial: 60.0 hours

## Fórmula de cálculo da classificação final
<p>Assessment in this course is distributed, without a final exam. There are two theoretical tests during the semester and one practical assignment with phased delivery that must be completed in groups of up to 3 students.<br /><br />The final grade is calculated as follows: 8 points for the practical assignment and 12 points for the tests.  Students must obtain a minimum grade of 25% in both the practical assignment and the tests to pass the course.</p>

## Obtenção de frequência
To pass the course, students must take at least one theoretical test and submit part of the assignment. Moreover, it is mandatory to earn a minimum grade of 25% both on the theoretical tests and on the practical assignment.

## Melhoria de classificação
Due to the distributed assessment format, there will be no supplementary or grade improvement exams.
