## Objetivos
<p>ENQUADRAMENTO</p>
<p>Na unidade curricular "Web Semântica e Dados Ligados" irão ser exploradas linguagens de anotação e de  interrogação e os princípios que lhes estão subjacentes, nomeadamente a lógica e a inferência. <br /><br />Estas linguagens e princípios fornecem um corpo de conhecimento que vai dos conceitos de Web Semântica e de Dados Ligados até à sua aplicação na descrição de recursos na Web e nas representações explícitas e interoperáveis para dados de domínios diversos. Estes conteúdos fornecem o suporte para objetivos de aprendizagem abstratos, que envolvem aspetos de modelação de dados, de criação de metadados e de desenvolvimento de ontologias. Os objetivos de aprendizagem de natureza aplicada seguem também os tópicos da unidade, nomeadamente os que têm a ver com o uso das tecnologias da Web Semântica, as linguagens de interrogação e os princípios dos Dados Ligados.<br /><br /><br />OBJETIVOS ESPECÍFICOS</p>
<p>1) Descrever e definir os conceitos e tecnologias associados à Web Semântica;</p>
<p>2) Analisar e preparar artefactos (p.ex.: ontologias) para utilização em soluções de Web Semântica;</p>
<p>3) Avaliar o valor e aplicabilidade de estratégias de Web Semântica em diversos contextos;</p>
<p>4) Identificar e aplicar várias ferramentas e técnicas relacionadas com a Web Semântica;</p>
<p>5) Analisar as caraterísticas de dados e documentos acessíveis a pessoas e a máquinas;</p>
<p>6) Relacionar os recursos na web com os metadados que os descrevem e relacionam;</p>
<p>7) Tratar as ontologias como fornecedoras de ferramentas de descrição;</p>
<p>8) Analisar ontologias existentes e criar novas ontologias;</p>
<p>9) Explorar aplicações que manipulam descrições de informação na web semântica e desenvolver métodos sistemáticos para criação de metadados;</p>
<p>10) Experimentar com aplicações que exploram a Web dos Dados Abertos Ligados;</p>
<p>11) Usar ferramentas e linguagens para explorar conteúdos da Web Semântica;</p>
<p>12) Comparar serviços baseados na web semântica e outras abordagens para a descrição de recursos.</p>

## Objetivos
<p>BACKGROUND</p>
The syllabus of Semantic Web and Linked Data has a strong emphasis on representation and querying languages and their underlying principles, namely logic and inference. <br /><br />These languages and principles provide a body of knowledge that ranges from the concepts of Semantic Web and Linked Data to their application in describing web resources and in explicit and interoperable representations for data in multiple domains. This content provides support for abstract learning objectives that involve aspects of data modeling, metadata creation, and ontology development. Applied learning objectives also follow the unit's topics, namely those on the use of Semantic Web technologies, interrogation languages, and Linked Data principles.<br />
<p>SPECIFIC OBJECTIVES</p>
<ol>
<li>Describe and define the concepts and technologies associated with the Semantic Web;</li>
<li>Analyze and prepare artifacts (e.g., ontologies) for use in Semantic Web solutions;</li>
<li>Evaluate the value and applicability of semantic web strategies in various contexts;</li>
<li>Identify and apply multiple Semantic Web-related tools and techniques;</li>
<li>Analyze the characteristics of data and documents accessible to people and machines;</li>
<li>Relate web resources to the metadata that describe and link them;</li>
<li>Treat ontologies as providers of description tools;</li>
<li>Analyze existing ontologies and create new ontologies;</li>
<li>Explore applications that manipulate semantic web information descriptions and develop systematic methods for creating metadata;</li>
<li>Experiment with applications that explore Linked Open Data on the Web;</li>
<li>Use tools and languages to explore Semantic Web content;</li>
<li>Compare semantic web-based services and other approaches to resource description.</li>
</ol>

<p>The theoretical components of classes are used for topic presentation, with reference to the bibliography, and for running small assignments to stimulate learning.
